# KURA — Bangalore

A sophisticated, high-end digital experience crafted for KURA Bangalore. This project acts as a luxury salon portfolio, built with modern web aesthetics in mind.

## Features
- **Three.js Integration:** Interactive 3D background components including floating sakura petals.
- **Micro-interactions:** Custom liquid blob cursor, asymmetrical layouts, and fluid cinematic transition effects.
- **Elegant Typography:** A curated blend of 'Shippori Mincho', 'Playfair Display', 'DM Mono', and 'Jost'.
- **Responsive Layout:** Adaptive design tailored for devices spanning from mobile to desktop.
- **Zero Dependencies:** A monolithic single-file architecture designed for fast loading and seamless, zero-framework execution (Three.js loaded via CDN).

## Deployment
This static site is configured for 1-click deployment on platforms like [Vercel](https://vercel.com) and [Netlify](https://netlify.com). It has been formatted to serve `index.html` as the root document out of the box.

## Local Development
To run this project locally, simply clone the repository and open `index.html` in your favorite browser. No node modules or build steps are required.

```bash
git clone https://github.com/twayibsmk/salon.git
cd salon
# Double click index.html or serve it via Live Server
```

## Tech Stack
- HTML5
- Pure CSS3 (custom variables & procedural animations)
- Vanilla JavaScript
- Three.js (via CDN)
